# MM_D-ND — Modello Duale Non-Duale

> **"Lo zero determina. Genera i due infiniti contrapposti — il primo dipolo."**

Repository canonico del framework matematico-ontologico **D-ND** (Duale Non-Duale).

## Contenuto

Questo repo contiene il modello fondazionale — la matematica, il metodo, il corpus e i paper. Non contiene codice eseguibile. Per il sistema operativo che implementa questo modello, vedi [THIA](https://github.com/GrazianoGuiducci/skill) (skill pubbliche).

### Il Kernel

Il file principale e' `kernel/KERNEL_MM_v1.md` — un system prompt di 14 sezioni che formalizza:

- **Proto-Assioma**: lo zero come entita' che determina
- **Dipolo Assiomatico**: la struttura duale che precede i poli
- **Assonanza/Dissonanza**: selezione automatica per coerenza
- **Risultante**: punto fisso autologico (R + 1 = R)
- **Potenzialita'**: P + A = costante, liberazione per movimento
- **Lagrangiana**: traiettoria di minima azione, massimo impatto

### Il Seme

`KERNEL_SEED.md` e' la forma invariante minima del Kernel — 9 assiomi + mappa di incarnazione per 5 ambienti (Claude, ChatGPT, Gemini, IDE, Cowork).
